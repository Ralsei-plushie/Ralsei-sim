# Ralsei-sim
My first 'game' made in Pygame!

### Controls
  * Arrow keys - Move around.
  * Hold H to hug.
  * Hold D to dance.

# Note 1
if you get an error while running the .exe there may be a few reasons with fixes
   1. You need to run pip install pygame
   2. You need to run pip install git+https://github.com/Ralsei-plushie/Pygame-easy.git
   3. If the /assets/ or /animations/ folder is not in the same folder as the .exe it will not run. (Look at Note 2 for more detail)
# Note 2
Reason for issue 1:
 This is built on Pygame, so it may be needed.
 This is made with the Pygame-easy wrapper, so you may need it installed. (Check my profile to see the actual project, a example is shown if you want to use it yourself!)
 The game looks in the same directory for the /assets/ and /animations/ folder, so if it can't find them, the game wont load.
